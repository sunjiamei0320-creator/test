# 📋 待办记录器

一个简洁、高效的待办事项管理工具，帮助你随手记录工作和生活中的任务。

## ✨ 功能特性

### 核心功能
- ✅ **快速记录** - 输入待办事项，按 Enter 或点击添加按钮
- ✅ **标记完成** - 勾选复选框标记任务为已完成
- ✅ **划掉展示** - 完成的任务会自动划掉
- ✅ **删除任务** - 支持删除不需要的待办事项

### 进阶功能
- 🏷️ **快速标签** - 一键添加优先级和分类标签
  - 🔴 紧急
  - 🟡 重要
  - 🟢 普通
  - 💼 工作
  - 📚 学习

- 📝 **备注功能** - 为每个任务添加详细的备注说明
- 🔍 **智能筛选** - 查看全部、待处理或已完成的任务
- 📊 **统计信息** - 实时显示总计、已完成、待处理数量
- 💾 **数据持久化** - 自动保存到本地存储
- 📤 **导出数据** - 导出为 JSON 格式备份
- 📥 **导入数据** - 从 JSON 文件导入任务
- 🗑️ **清除已完成** - 一键清理已完成的任务

## 🚀 快速开始

1. **打开应用** - 在浏览器中打开 `index.html` 文件
2. **输入任务** - 在输入框中输入你的待办事项
3. **添加任务** - 按 Enter 键或点击"添加"按钮
4. **标记完成** - 完成后勾选复选框
5. **添加备注** - 点击 📝 按钮添加任务备注
6. **管理标签** - 点击快速标签或右键菜单管理标签
7. **导出备份** - 点击"导出数据"保存任务列表

## 🎯 使用技巧

### 快速添加标签
- 点击快速标签按钮（如 🔴 紧急），会自动附加到当前输入框
- 组合使用多个标签：`完成报告 🔴 紧急 💼 工作`

### 添加备注
- 点击每个任务旁的 📝 按钮
- 输入备注内容并确认
- 备注会显示在任务下方，便于查看详情

### 筛选任务
- **全部** - 显示所有任务
- **待处理** - 只显示未完成的任务
- **已完成** - 只显示已完成的任务

### 数据管理
- 数据自动保存到浏览器本地存储
- 可以随时导出数据作为备份
- 可以从备份的 JSON 文件导入任务
